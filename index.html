pdf-lib Overview

pdf-lib is a JavaScript library for creating, modifying, and working with PDF documents. This index provides a quick reference to its core components.

1. Core Concepts

  • PDFDocument: The central class representing an entire PDF document. You'll use this to create new PDFs or load existing ones.
  • PDFPage: Represents a single page within a PDF document. You draw content onto pages.
  • PDFFont: Represents a font used for rendering text.
  • PDFImage: Represents an image embedded in the PDF.
  • PDFForm: Represents the interactive form within the PDF.
  • PDFField: Base class for all Form Fields.

2. Enums (Predefined Choices)

  • BlendMode: How colors are blended together.
  • ColorTypes: Defines the color type.
  • Duplex: Printing options for double-sided printing.
  • ImageAlignment: Defines the image alignment within its bounds.
  • LineCapStyle: How the end of a line is drawn (e.g., rounded, square).
  • LineJoinStyle: How lines are joined together (e.g., rounded, mitered).
  • NonFullScreenPageMode: How the PDF viewer should display pages when not in full-screen mode.
  • ParseSpeeds: Defines the speed at which the PDF is parsed.
  • PrintScaling: Defines the way the PDF is scaled while being printed.
  • ReadingDirection: Defines the direction the text is being read in.
  • RotationTypes: Defines the rotation type.
  • StandardFonts: A list of commonly available fonts (e.g., Helvetica, Times-Roman).
  • TextAlignment: How text is aligned (e.g., left, center, right, justify).
  • TextRenderingMode: Defines how the text is rendered.

3. Classes (Objects)

  • PDFButton: Represents a button field in a PDF form.
  • PDFCheckBox: Represents a checkbox field in a PDF form.
  • PDFDropdown: Represents a dropdown field in a PDF form.
  • PDFEmbeddedPage: Represents a page being embedded in another PDF.
  • PDFOptionList: Represents an OptionList field in a PDF form.
  • PDFRadioGroup: Represents a radiogroup field in a PDF form.
  • PDFSignature: Represents a signature field in a PDF form.
  • PDFTextField: Represents a text field in a PDF form.
  • ViewerPreferences: Represents the Viewer Preferences.
